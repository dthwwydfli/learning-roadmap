# learning-roadmap
A personal, interactive learning tracker built as a single-page HTML app. Covers everything needed to go from learning JavaScript at university to shipping a real-world project — broken down week by week, with progress that saves automatically in the browser.

Live website: https://dthwwydfli.github.io/learning-roadmap/

Why I Built this:
Me and a classmate are building an ASL (American Sign Language) recognition app using React, TensorFlow.js, MediaPipe, and the Anthropic API. Before jumping into the project, we needed a structured plan to get from JavaScript basics to being comfortable enough to ship something real.
This roadmap is that plan — built as an actual app so we can track progress and refer back to it throughout the project.

What it covers:
The roadmap is split into 7 sections, each with weekly task breakdown:
- JavaScript
- Git & Github
- npm
- React
- AI tools
- Tailwind CSS
- ASL project

Features:
Tabbed navigation — each topic is its own page, no scrolling through everything at once
Weekly breakdown — tasks are grouped by week so it's clear what to focus on
Progress tracking — every task is a checkbox; progress bars update in real time
Saves automatically — uses localStorage so progress persists between sessions
roadmap.sh reference — links to the full roadmap.sh/frontend guide at the top of every page

Built With AI:
This project was designed and built with the help of Claude (Anthropic). The roadmap structure, weekly task breakdowns, and the HTML/CSS/JS for the app itself were all generated through a conversation where I described what I needed and iterated on the output.
This is intentional — learning to use AI tools effectively is itself one of the skills on this roadmap. Being able to direct an AI to build something useful, evaluate the output, and make decisions about what to keep or change is a real skill, and one that comes up in interviews.
